# Modern WMS System

프로젝트 경험을 바탕으로 WMS 출고 시스템을 현대적인 Spring Boot 스택으로 재설계한 프로젝트입니다.

## 배경
- 실무에서 레거시 프로시저 기반 WMS를 경험
- 현대적인 Java/Spring 생태계로 리팩토링한다면? 이라는 질문에서 시작
- 도메인 주도 설계 및 테스트 주도 개발 적용

## 주요 기능
- 출고 오더 관리
- 재고 선택 전략 (FIFO)
- Twin 작업 판단 로직
- 자동 랙이동 결정

## 기술 스택
- Java, Spring Boot
- JPA, QueryDSL
- JUnit 5

⚠️ 본 프로젝트는 개인 학습 목적으로 작성되었으며, 
   실제 업무 코드와는 무관합니다.
